## Quick start

```shell
docker-compose up -d
```

Terminal 1:
```shell
cd packages/jukboks-backend
npm i
npm run dev
```

Terminal 2:
```shell
cd packages/jukboks-frontend
npm i
npm run dev
```

Navigate to http://localhost:3000
